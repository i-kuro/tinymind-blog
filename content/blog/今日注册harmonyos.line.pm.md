##在Github创建Tinymind博客

原理是创建一个仓库用于存储文本&图片，每次提交 Tinymind 就会读取最新数据，然后渲染在网页上。

具体操作方法：

1、授权 Tinymind 访问您的公共 GitHub 仓库。
2、Tinymind 在您的 GitHub 账户中创建一个名为 "tinymind-blog" 的公共仓库。
3、您在 Tinymind 上写的任何内容都会提交到这个仓库。
4、您的数据存储在您的 GitHub 仓库中，与 Tinymind 分开。
